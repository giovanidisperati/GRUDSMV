# **Aula 03 - Introdução às APIS REST e ao Spring Boot**  

Nesta aula apresentaremos uma introdução aos conceitos basilares para construção de uma API REST para gerenciar uma lista de contatos utilizando **Spring Boot**. 

A ideia é demonstrar, passo a passo, como criar um projeto **Spring Boot** e expor métodos que permitem a criação, consulta, atualização e exclusão de contatos (operações conhecidas como CRUD). Antes disso, entretanto, é preciso definir alguns conceitos principais: o que são APIs, o que é REST, quais verbos HTTP vamos .
